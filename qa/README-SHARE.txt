Sitka × Shopify — technical scenarios microsite
==============================================

How to view
-----------
1. Unzip this folder if needed. Keep everything together (index.html and the assets/ folder must stay side by side).

2. Open index.html:
   - Double-click index.html, or
   - Right-click → Open With → Chrome / Safari / Edge

3. If images or fonts look broken when opened as a file:// URL, run a tiny local server from this folder:
     python3 -m http.server 8080
   Then visit http://localhost:8080 in your browser.

What’s inside
-------------
- index.html — single-page site (sections 1–4)
- assets/ — screenshots, logos (section1 … section4, shopify-logo.svg, sitka-logo.png)

No build step or npm install required.
